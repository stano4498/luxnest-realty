# LuxNest Realty

Welcome to **LuxNest Realty** — a premium, luxurious real estate website designed to connect buyers, sellers, and investors with high-end properties and expert agents. This repository contains the full source code for a professional, multi-page real estate platform crafted with HTML, CSS, and JavaScript.

---

## 🚀 Features

- **10 Fully Developed Pages** including:
  - **Home:** Engaging landing page presenting the agency’s proposition and services.
  - **Property Listings:** Displaying 15 detailed property cards with images and specs.
  - **Search:** Robust search filters by location, price, type, bedrooms, and bathrooms.
  - **Blog:** Educational articles for newcomers to real estate.
  - **FAQ:** Common questions answered for user convenience.
  - **Financial:** Information on loan options plus an interactive mortgage calculator.
  - **About Us:** Agency history, expertise, and professional team details.
  - **Community:** Neighborhood insights including schools, amenities, and local attractions.
  - **Contact:** Contact form with embedded Google Maps for location.
  - **Testimonials:** Six glowing client reviews highlighting positive experiences.
  - **Buying & Selling Guides:** Downloadable PDF guides to help users make informed decisions.

- **Responsive Design:** Using modern CSS Grid, Flexbox, and media queries for seamless experience across all devices.
- **Luxurious Theme:** Sleek color palette with consistent navbar and footer styling.
- **Interactive JavaScript Features:** Mortgage calculator, property search filtering, and form validation.
- **Social Media Integration:** Icons and links included in the footer.
- **User Login Placeholder:** Ready for future user authentication enhancements.
